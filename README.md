## 📁 Проекты

### 🔗 URL-шортер
[→ Репозиторий](https://github.com/OlegRozh/url-shortener)

**Сделано:**
REST API на Go для сокращения ссылок (кастомные алиасы, автогенерация, редирект), обработка конфликтов. Настроены сбор логов (Loki) и метрик (Grafana) через Docker Compose.

- Время ответа API < 50 мс.
- Обработка до 1000 запросов/с локально.
- Время обнаружения проблем — сокращено до 5 минут (против часов без observability).

**Сложности и решения:**
- Конфликты алиасов → реализовал автоматическую перегенирацию в случае дубля
- Горячий перезапуск контейнеров → добавил health-checks в Docker Compose
- Утечка памяти при нагрузке → профилировал через pprof, оптимизировал пул соединений

**Приносит пользу:**
- Упрощает трекинг ссылок для маркетинга и соцсетей.
- Ускоряет дебаг и мониторинг.
- Демонстрирует production‑ready подход (контейнеризация, чистая архитектура).
